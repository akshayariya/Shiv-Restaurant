# Shiv Restaurant

A clean and user-friendly online menu for Shiv Restaurant. Browse our offerings and discover delicious dining options with an intuitive web interface.

## Features

- 📋 **Easy-to-Browse Menu** - Simple and organized menu display
- 📱 **Responsive Design** - Works seamlessly on desktop and mobile devices
- ✨ **Clean Interface** - Intuitive user experience
- 🎨 **Professional Styling** - Modern and attractive presentation

## Technologies Used

- **HTML** (91.7%) - Structure and content
- **CSS** (8.3%) - Styling and layout

## Getting Started

### Prerequisites

All you need is a web browser to view the online menu.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/akshayariya/Shiv-Restaurant.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Shiv-Restaurant
   ```

3. Open `index.html` in your web browser to view the menu.

## Usage

Simply open the restaurant menu in your browser to:
- Browse menu items
- View descriptions and pricing
- Get restaurant information
